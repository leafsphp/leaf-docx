Leaf PHP 1.2.0 Create clean, simple but powerful web applications and APIs quickly.

Get Started Checkout v2.0.0
// Sample index.php require_once __DIR__ . '/vendor/autoload.php'; $leaf = new Leaf\Core\Leaf(); $response = new Leaf\Core\Response(); $request = new Leaf\Core\Request(); $leaf->get('/book/{id}', function($id) use($response) { echo $response->respond(["id" => $id]); }); $leaf->get('/home', function() use($response) { echo $response->renderHtmlPage('./index.html'); }); $leaf->post('/books/add', function() use($request, $response) { $title = $request->getParam('title'); echo $response->respond(["message" => "hello"]); }); $leaf->run();

Quickly Create PHP Projects

Leaf is a PHP micro-framework which helps you create web applications and APIs from the ground up in minutes.

Structured PHP

PHP is a free language, a language with only a handful of rules.

This freedom gives rise to a lot of problems for developers since there's usually many ways to achieve the same thing.

Leaf gives you a particular structure to follow whiles maintaining PHP's freedom.

// simple structure $leaf->get('/', function() use($response) { return $response->respond(["message" => "hello"]); }); $leaf->post('/books/add', function() use($request, $response) { $title = $request->getParam('title'); return $response->respond(["message" => "hello"]); });

Simple PHP methods

Leaf PHP provides you with very simple methods to handle various situations.

With a clean structure and simple code, it's super easy to master Leaf's API.

// get request params $leaf->post('/name/add', function() use($request) { $name = $request->getParam('name'); });

Clean, Fast, Reliable

Leaf PHP uses only components written by leaf in raw, tested PHP.

This means that in terms of performance, Leaf is the fastest PHP framework.

Leaf also uses clean, minimal code to achieve complex results.

// generate JWT token $leaf->post('/token/create', function() use($response, $authentication) { $token = $authentication->createSimpleToken(22, 'Secret key'); return $response->respond(["token" => $token]); });
