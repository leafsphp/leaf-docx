Leaf Projects
Projects built from and for Leaf

Leaf MVC

Leaf MVC

Contribute to Leaf MVC Developments

Leaf MVC is a simple framework with Leaf PHP at it's core. Leaf MVC basically offers Leaf PHP framework in an MVC environment.

Visit Leaf MVC Repo

Leaf API

Contribute to Leaf MVC Developments

Leaf API is a simple framework with Leaf PHP at it's core. Leaf API is lightweight and is made customly for API development. Leaf API tries to use an MVC-like structure to give to give developers a good experience. We call this Model Request Response Controller(MRRC).

Visit Leaf API Repo

Leaf Veins

Contribute to Leaf Veins

Leaf Veins is the official templating engine of the Leaf PHP framework, it comes pre-packaged but is a package on it's own. Veins try to keep the feel of writing normal mark-up with code like {if} {else} {/if}

Visit Leaf Veins Repo
