Get Started View Leaf Projects
require_once __DIR__ . '/vendor/autoload.php'; $leaf = new Leaf\Core\Leaf; $response = new Leaf\Core\Http\Response; $request = new Leaf\Core\Http\Request; $db = new Leaf\Core\Db\Mysqli; $db->connect("host", "username", "password", "dbname"); $leaf->get('/book/{id}', function($id) use($response) { $response->respond(["id" => $id]); }); $leaf->get('/home', function() use($response) { $response->renderPage('./index.html'); }); $leaf->post('/books/add', function() use($request, $response, $db) { $title = $request->get('title'); $db->insert("books", "title", "?", [$title]); $response->respond(["message" => "$title added"]); }); $leaf->run();

What's new in v1.5.0

Leaf Database Tool

Leaf's "simple query builder" provides a convenient but usual way to quickly create and run database queries. It can be used to perform most database operations in your app.

Leaf's "simple query builder" currently supports Mysqli and PDO connections, though we still recommend using Mysqli. There's no need to worry about SQL injection as parameter binding is also supported and easy to use😉💪

View Leaf DB' docs here

$leaf = new Leaf\Core\Leaf; $response = new Leaf\Core\Http\Response; $db = new Leaf\Core\Db\Mysqli; $db->connect("host", "username", "password", "dbname"); $leaf->get('/posts/all', function() use($db, $response) { # get all rows from "posts" table $posts = $this->select("posts")->fetchAll(); $response->respond($posts); }); $leaf->run();

Leaf Simple Auth

Setting up a login and sign-up system can sometimes be a very unpleasant experience, especially for new devs. For pro devs, the challenge is "the standard". For this, Leaf has prepared something really simple.

Leaf Simple Auth allows you to set-up authentication in just one line of code😅😅. Literally 1 line😎.

View auth docs

$leaf = new Leaf\Core\Leaf; $response = new Leaf\Core\Http\Response; $leaf->post("/auth/register", function() use($response, $auth) { $username = $auth->get("username"); ... $auth->basicRegister($username, $email, $password, $confirm_password); $response->respondWithCode( $auth->basicLogin($username, $password), 200); });

New Form Validate method

Leaf now provides a much simpler way to validate a parameter using Leaf Forms. In one line, you can create a bunch of rules to validate a parameter with. Validate simply makes sure that the passed selected parameters pass these validation tests. It's never been simpler😎

Parameters which fail the form validation are saved in the form's errors which can be accessed with errors()

View Form docs

$leaf = new Leaf\Core\Leaf; $response = new Leaf\Core\Http\Response; $form = new Leaf\Core\Form; $leaf->get("/login", function() use($form, $response) { $form->validate([ "username" => "validUsername", "email" => "email", "password" => "required" ]); !empty($form->errors()) ? $response->respond($form->errors()) : null; });
